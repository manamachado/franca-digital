# Exemplo de um arquivo HTML

```html
<!DOCTYPE html>
<html lang="pt-br">

<head>
  <meta charset="UTF-8">
  <title>Meu primeiro HTML</title>
</head>

<body>
  <!-- TÍTULOS -->
  <h1>Titulo H1</h1>
  <h2>Título H2</h2>
  <h3>Título H3</h3>
  <h4>Título H4</h4>
  <h5>Título H5</h5>
  <h6>Título H6</h6>

  <!-- PARÁGRAFO COM QUEBRA DE LINHA -->
  <p>Lorem ipsum, dolor sit amet consectetur adipisicing elit. <br>Suscipit repellat alias quod quidem laboriosam porro,
    quia cumque, autem hic illo nam ipsam. <br>Quod porro, consequuntur quo quidem doloremque aliquid nisi.</p>

  <!-- LINK ABRINDO NA MESMA JANELA -->
  <a href="https://magazineluiza.com.br">Exemplo de link abrindo na mesma janela</a>

  <!-- QUEBRA DE LINHA -->
  <br>

  <!-- LINK ABRINDO EM OUTRA ABA -->
  <a href="https://magazineluiza.com.br" target="_blank">Exemplo de link abrindo em outra aba</a>

  <!-- DUAS QUEBRAS DE LINHA -->
  <br>
  <br>

  <!-- TABELA -->
  <table border="1">
    <!-- <tr> DEFINE UMA LINHA NA TABELA -->
    <tr>
      <!-- <th> DEFINE UMA CÉLULAR DE HEADER NA TABELA -->
      <th>Nome</th>
      <th>Endereço</th>
      <th>Telefone</th>
    </tr>
    <tr>
      <!-- <td> DEFINE UMA CÉLULA PADRÃO NA TABELA -->
      <td>João</td>
      <td>Av. Brasil, 100</td>
      <td>(16) 9999-9999</td>
    </tr>
    <tr>
      <td>José</td>
      <td>Av. Brasil, 297</td>
      <td>(16) 9999-9999</td>
    </tr>
    <tr>
      <td>Maria</td>
      <td>Av. Brasil, 297</td>
      <td>(16) 9999-9999</td>
    </tr>
  </table>

  <!-- LISTA NÃO ORDENADA -->
  <ul>
    <!-- <li> DEFINE UM ITEM DA LISTA -->
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </ul>

  <!-- LISTA ORDENADA -->
  <ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </ol>

</body>

</html>
```
