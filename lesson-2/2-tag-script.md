# Tag script

O HTML possui uma tag especial que nos permite escrever código Javascript dentro dela, é a tag `<script>`.

## Exemplo

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Olá mundo no console</title>
</head>
<body>
  <script>
    Aqui vai o código Javascript
  </script>
</body>
</html>
```
